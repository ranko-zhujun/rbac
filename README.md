# 安装步骤
1. 复制install目录下的dev文件至config目录下
2. 修改dev目录下的db.php中的数据库配置
3. 数据库导入SQL脚本,对应的脚本install目录下
4. 默认密码：ranko/admin
5. 后台地址：index.php?r=site/login
